  • showMenuHeader

    This flag will affect if there will be visible header image in Side Menu. This header will be visible only if no promotions are set. Promotion headers have higher priority than this flag and even this flag will be set tot true, the menu header will not be shown if there is promotion present. To show your custom header image, add Image Asset to your app’s bundle and name the asset MASHeader
